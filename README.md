# 🚆 RailSense AI

AI-powered railway traffic prediction and visualization system.

## 🔥 Features
- ML-based congestion prediction
- Interactive dashboard (React)
- Real-time analytics
- Prediction confidence scores

## 🛠 Tech Stack
- Frontend: React, Tailwind, Recharts
- Backend: Flask, Pandas, Scikit-learn
- ML Model: Random Forest

## 🚀 Run Locally

### Backend
cd backend
python3 app.py

### Frontend
cd train-ai-ui
npm install
npm start
